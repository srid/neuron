Code Syntax Highlighting

Neuron provides syntax highlighting through Prism, a JavaScript library that highlights code blocks in the web browser. To customize Prism’s theme, or to provide your own syntax highlighter, you can do so in head.html (see Custom JavaScript and CSS).

To activate syntax highlighting, you must specify the language in your fenced code blocks (see example here) or view source of the below.

Demo

Haskell:

module Prime where 

primes = filterPrime [2..]
  where 
    filterPrime (p:xs) =
      p : filterPrime [x | x <- xs, x `mod` p /= 0]
